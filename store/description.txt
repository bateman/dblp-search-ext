Search academic publications on dblp.org from everywhere. Type a title in the search box or highlight any text on a web page, open the extension, and instantly see the matching publications — then copy or download their BibTeX entries.

# Features
- Search publications by typing a title or by highlighting text on any page first.
- Right-click selected text on any page to search it on dblp or resolve a DOI, without opening the popup.
- Results are shown in a sortable table (type, title, authors, year, venue, DOI, open/closed access): sort by authors, year, or venue, restore the default order via the title column, and filter by publication type.
- Entries are color-coded by type (proceedings, books, journals, etc.), matching the dblp.org scheme.
- Copy button: copy an entry's BibTeX reference to the clipboard.
- Download button: save an entry as a .bib file, named after its citation key.
- Keyboard navigation: move between results with the up/down arrows, then press C to copy BibTeX, D to download it, B to open the dblp page, or O to open the DOI.
- Customizable citation keys: replace dblp's default key with your own format using a drag-and-drop builder (Author, Year, Venue, Title, with dash or underscore separators); optionally capitalize the author name and uppercase the venue (e.g., Calefato2023ESEM, calefato_2023_esem).
- Clean up BibTeX: optionally remove the timestamp, biburl, bibsource, and url fields.
- Pagination with Previous/Next, current page, total pages, and result count.
- Search state (query and results) is saved locally, so you can leave and come back without losing them.
- Useless duplicate CoRR abs entries are filtered out automatically.
- Powered by the official dblp.org API for fast, reliable queries.

# Options
- Popup width: set the default popup width (500-800 px); narrower widths hide some table columns automatically.
- Maximum number of results: default 30, capped at 1000.
- BibTeX key renaming: enable and customize the citation key format with the drag-and-drop builder, plus author capitalization and venue uppercasing.
- Remove timestamp, biburl, and bibsource fields from copied/downloaded entries.
- Remove the URL field from copied/downloaded entries.

# Privacy
The extension sends search queries to dblp.org and fetches public BibTeX records from it. Copying writes the entry to your clipboard; downloading saves it as a .bib file on your device. Settings and recent search state are stored locally in your browser. No personal data is collected or sent to the developer.
